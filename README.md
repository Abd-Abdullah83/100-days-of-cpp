## 📌 Highlights So Far

### Day 001 — Fraction Class
A complete `fraction` class featuring:
- ✦ Full **Rule of Five** (copy, move, destructor)
- ✦ All **arithmetic operators** (`+` `-` `*` `/`)
- ✦ **Integer-fraction** mixed arithmetic (`2 * frac`, `frac + 3`)
- ✦ All **comparison operators** using cross-multiplication
- ✦ **GCD-based simplification** after every operation
- ✦ **Division-by-zero** protection
- ✦ Stream operators (`<<` and `>>`)
